# Fashion E-Commerce Tutorial for Students

Welcome to the Fashion E-Commerce Tutorial! This comprehensive course will teach you how to build a **professional, production-ready** fashion e-commerce website from scratch using modern, industry-standard tools.

## 🎯 What You'll Build

A fully functional, **professional-grade** fashion e-commerce website with:
- Product catalog with images and details
- Search, filter, and advanced discovery features
- Shopping cart with persistent storage
- Professional checkout with Cash on Delivery
- User authentication and order tracking
- Wishlist and product reviews
- **Professional admin dashboard** with analytics
- Email notifications and SEO optimization
- **Beautiful UI with shadcn/ui components**
- Fully responsive design for all devices
- Progressive Web App (PWA) capabilities
- Live deployment

## 🛠️ Technologies You'll Learn

- **Next.js 14** - Modern React framework
- **React** - JavaScript library for building user interfaces
- **Strapi CMS** - Headless content management system
- **NextAuth.js** - Authentication for Next.js
- **shadcn/ui** - Professional UI component library
- **Tailwind CSS** - Utility-first CSS framework
- **Radix UI** - Accessible component primitives

## 🚀 Getting Started

**Important:** For a professional-looking website, start with shadcn/ui!

1. **[Day 0: shadcn/ui Setup](./day-00-shadcn-setup.md)** - Set up professional UI components (30 min)
2. **[Day 1: Introduction \u0026 Environment Setup](./day-01-setup.md)** - Install Next.js and Strapi
3. Follow each day's tasks in order
4. Don't skip days - each builds on the previous!

### Quick Start Path

```bash
# 1. Create Next.js project
npx create-next-app@latest frontend --js --tailwind --app

# 2. Set up shadcn/ui (makes it professional!)
cd frontend
npx shadcn-ui@latest init

# 3. Install essential components
npx shadcn-ui@latest add button card input label badge dialog table tabs

# 4. Start building!
```
